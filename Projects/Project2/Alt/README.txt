Import Jar File.

Run the program. At prompt, enter a full 
website (ex: google.com should be "http://www.google.com" 
including the "http://" and "www.").

Next prompt inquires how many pages to go though,
though practice, more than 5 pages seems to take a long
time. Use at own risk.

After reading is complete, the user will be informed 
how long it took.

Links will be saved in "links.txt" and email addresses 
will be saved in "emails.txt".
