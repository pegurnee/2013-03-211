Import Jar File.

The file input takes place in the HelpFrame file, 
if you wish to use simply the HelpFrame, 
feel free to only load and run HelpTester.

Otherwise HelpFrame can be accessed from the User Manual 
menu item from the help menu on the menu bar.

From the HelpFrame, use the Search Help button to look up a search term,
if there are more than 5 search results, you can refine the search by 
searching for another term. Results will appear in the yellow box on the left.

Next and Previous Search Result buttons are used to scroll through the results.

Browse Help will allow the user to view all items in the 
GLOSSARY.txt.

Reload Help reloads the GLOSSARY.txt file including all 
new changes.
